## 📦 Installation

```bash
curl -fsSL https://github.com/hieudoanm/qrcode/raw/refs/heads/master/packages/cli/bin/qrcode -o qrcode
chmod +x qrcode
sudo mv qrcode /usr/local/bin/
```

## 🚀 Usage

```bash
qrcode
qrcode generate
```
